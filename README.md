How to run project ?

    1. Run yarn install
    2. Run yarn start

How to run unit and integrated tests?
-> Run: yarn test

How to run E2E tests?

    1. Run: yarn cypress open (Before you should run: yarn start)
    2. In Cypress window click E2E testing
    3. Select browser
    4. Run specific test scenario e.g. heroFilter.cy.ts
